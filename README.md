# GameUp
A gamified coaching mobile app developed for passionate players 

IMPORTANT: To run this app, you must download Expo Go on either IOS or Android, run the base code through Terminal by using commads such as npm install, then npx expo start.
All the Terminal steps must be done while inside the reposity of the porject, in this case it should be cd GameUp-1\GameUp\GameUp\hazardous-green-apples\hazardous-green-apples.
Once the code has ran through the Terminal after entering the npx expo start line, there will be a QR code presented, scan the QR code with the Expo Go app, then the actual
application will run
